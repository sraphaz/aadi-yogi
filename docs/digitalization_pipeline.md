# Digitalization Pipeline

The future pipeline for physical and semi-structured materials is expected to follow this sequence:

```text
physical book
-> scan
-> OCR
-> OCR correction
-> Markdown conversion
-> metadata
-> review
-> chunking
-> indexing
-> agent availability
```

## Review Principles

- keep raw scans and OCR artifacts separate from curated Markdown
- document editorial interventions
- review metadata before downstream indexing
- preserve provenance from physical source to final chunk
