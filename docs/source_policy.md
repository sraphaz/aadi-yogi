# Source Policy

Every source entry should be classified clearly so later ingestion and retrieval systems can apply the right handling rules.

## Source Categories

- `primary_text`: canonical or primary spiritual source material
- `commentary`: traditional or modern commentary tied to a source
- `translation`: translated rendering of a source text
- `modern_interpretation`: spiritually informed modern exposition
- `academic_source`: scholarly or historical source
- `internal_note`: repository-only operational or editorial note
- `synthesis_note`: cross-source synthesis created inside this repository

## Policy Notes

- Primary material requires stronger citation and copyright handling.
- Commentary and interpretation must not be mistaken for original scripture.
- Internal and synthesis notes must never be presented as canonical authority.
