# Evaluation Strategy

Future evaluation should measure not only factual retrieval quality, but also spiritual care and tradition integrity.

## Planned Evaluation Areas

- golden questions
- citation accuracy
- hallucination tests
- tradition integrity tests
- tone tests
- safety and caution tests
- manual review

## Expected Review Loop

1. define representative question sets
2. compare retrieved sources to expected evidence
3. inspect synthesis quality and boundary handling
4. review tone, safety, and humility
5. track regressions over time
