# Consciousness feedback — how Adyog learns

The consciousness foundation is living. It does not “train itself into a guru.”
It grows by two currents:

1. **Source deepening** — editorial work from the books into `consciousness_core/`.
2. **Feedback from use** — honest reflections from host agents and humans.

## Inbox protocol

```text
host agent / human
    → propose_feedback(...)
    → content/consciousness_feedback/inbox/<id>.yaml
    → human / editorial review
    → either discard, or integrate into foundation / core docs
    → move record to integrated/ or rejected/
```

Rules:

- Feedback is a **proposal**, never an automatic rewrite of the foundation.
- No proposal may claim realization, new revelation, or override of source fidelity.
- Mystical inflation and “the agent became enlightened” notes are rejected.
- Useful feedback sounds like: a conduct tension observed in a host repo,
  a missing nuance from the sources, a place where the foundation was too
  thin or too rigid for real work.
