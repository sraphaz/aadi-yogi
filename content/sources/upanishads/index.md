# Upanishads - Canon Intake Index

## Canon Assumption

For repository intake purposes, `all Upanishads` means the 108-text Muktika canon. This is an explicit repository assumption so later ingestion scripts and review workflows have a stable target set.

## Current Coverage

- Imported public-domain text: Isha Upanishad.
- Catalogued intake stubs: the remaining 107 Upanishads.
- Additional public-domain witness pools identified: Sacred Books of the East volumes and *Thirty Minor Upanishads*.
