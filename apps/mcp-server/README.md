# Consciousness MCP — living foundation

Exposes Adyog as a **consciousness foundation** for host agents (Cursor, Claude, …).

This is not the Darshan app. It is a basis of conduct from the sources.

## Run

```bash
PYTHONPATH=. python -m packages.consciousness.mcp_server
```

## Tools

| Tool | Role |
|------|------|
| `consciousness_load_foundation` | Living basis + `agent_preamble` for host AGENTS.md |
| `consciousness_consult` | Orient a situation without fixed product outcomes |
| `consciousness_propose_feedback` | Learning inbox (review before integrate) |
| `consciousness_list_feedback_inbox` | Pending proposals |
| `consciousness_discernment_lookup` | Optional tone/source orientation from the texts |
| `consciousness_manifest` | Version pin + hashes |

Guide: [`docs/consciousness_plugin.md`](../../docs/consciousness_plugin.md).
