# Web App

This folder is reserved for the future web interface that will present source-grounded guidance, citations, and study flows to end users.

The interface concept and specification are defined:

- `docs/darshan_interface_concept.md`: the Darshan concept - design principles and the seven gestures.
- `docs/darshan_interface_spec.md`: screens, flows, response contract, presence metrics, build phases.
- `docs/skyforge/darshan/`: Sky-Forge session package (brief, RF/NFR, UX spec, SKY indices, maturity).

No application code is implemented yet.
